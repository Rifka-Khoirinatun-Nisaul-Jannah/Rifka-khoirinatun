Tugas Kelompok

Web Server Dinamis

Latar Belakang Projek: Mengubah halaman index.html dengan halaman yang dibuat sendiri (team), sehingga dari sisi client dapat mengakses halaman website tersebut dengan sukses via arsitektur server terintegrasi.

Berbeda dengan server statis yang hanya mengirimkan file mentah apa adanya, implementasi **Web Server Dinamis** ini melibatkan skrip pemrosesan logika di sisi backend. Hal ini memungkinkan validasi data secara *real-time*, proteksi routing, manajemen sesi pengguna (*user session*), serta pengiriman data asinkron berbasis API jembatan komunikasi jaringan.
